# Software Craftsmanship

Software craftsmanship emphasizes code quality, continuous learning, and technical responsibility.

## Resources and metaphors

### Stone Soup

- Start from a simple core
- Build progressively with the group's contributions
- Foster engagement around a first concrete result

### Frog Soup

- Repeated small compromises can damage a system without triggering an immediate alarm
- You must maintain active vigilance over technical debt

## What it means in practice

- Write readable code
- Seek short feedback loops
- Automate validation
- Embrace rigor in specifications and tests
